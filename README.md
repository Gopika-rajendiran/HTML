# HTML
## Description

This is a simple HTML project that displays an **Upcoming Events** webpage. It is designed for beginners who are learning HTML and demonstrates the use of basic HTML elements to create a clean and organized webpage.

The page includes:
- A webpage title and favicon
- A heading with an introduction
- Event details with images
- Clickable links
- Contact information in the footer

## Features

- 📅 Upcoming Events section
- 🖼️ Event images
- 🔗 Clickable hyperlinks
- 📱 Simple and responsive layout
- 📞 Contact section
- 🎨 Black background with white text for better readability

## Technologies Used

- HTML5
